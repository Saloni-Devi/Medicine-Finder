**Medicine Finder 💊**
**🔴 Live Demo:** https://saloni-devi.github.io/Medicine-Finder/

### Tech Stack:
- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API
- Git & GitHub Pages
  
**Features:**
- Search medicines on homepage
- Detailed view on details.html
- Watchlist with Local Storage
- Mobile responsive design

- Building multi-page websites without frameworks
- Responsive design using CSS Flexbox & Media Queries
- Version control with Git & deploying on GitHub Pages
  
### How to Run:
1. Clone the repo
